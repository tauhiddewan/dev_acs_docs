Connecting to the Clusters
==========================

Triton Access
^^^^^^^^^^^^^

- **DNS:** ``t2.idsc.miami.edu``
- **Access:**  
  Connect via :ref:`ssh` over secure University of Miami networks.  
  For graphical sessions, use :ref:`x11`
- **Credentials:**  
  Requires a valid **IDSC Account**.

Pegasus Access
^^^^^^^^^^^^^

- **DNS:** ``pegasus2.ccs.miami.edu``
- **Access:**  
  Connect via :ref:`ssh` over secure University of Miami networks.  
  For graphical sessions, use :ref:`x11`
- **Credentials:**  
  Requires a valid **IDSC Account**.
